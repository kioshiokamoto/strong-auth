# Strong Auth (WIP)

Aplicación de registro, inicio de sesión y perfil de usuario con JWT


## Tech Stack

**Backend:** Nest, TypeORM, Postgres

## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/strong-auth
```

Ir a ruta de proyecto

```bash
  cd strong-auth
```

Instalar dependencias

```bash
  npm install
```

Iniciar servidor

```bash
  npm run start
```
